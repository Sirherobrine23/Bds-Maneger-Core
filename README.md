# Bds Maneger Core

A quick and simple way to create and manage Minecraft servers like `Bedrock`, `Java`, `Paper`, `Spigot`, `Pocketmine-MP` and `Powernukkit`.

## Requeriments

- Nodejs (`16^`).
- linux/Windows/Macos.
